# Helico

An AlphaFold3 clone, built from scratch in PyTorch for experimentation.

## What's here

- **[Experiments](experiments/index.md)** — one page per research experiment. Every experiment is preregistered as a GitHub issue, executed as a Jupyter notebook on a branch (or on `main`), and its analysis lives in the notebook itself.
- **[Benchmarks](benchmarks.md)** — cross-experiment FoldBench comparison tables.
- **[Releases](releases.md)** — officially released model checkpoints on HuggingFace.
