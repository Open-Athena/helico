# Released models

Official Helico checkpoints live at
[timodonnell/helico](https://huggingface.co/timodonnell/helico) on HuggingFace.
Each release is tagged on GitHub and published with a model card documenting:

- Training configuration
- Data snapshot
- FoldBench scores across all 9 categories
- WandB run link
- Git SHA

Experimental checkpoints (auto-published for every experiment run) live at
[timodonnell/helico-experiments](https://huggingface.co/buckets/timodonnell/helico-experiments)
and are not part of the release cadence.
